# Tensorflow-Lite-with-ESP32CAM

###### Tools <br />
1. Python 3.6 <br />
2. Install urllib <br />
3. Library ESP32CAM <br />

###### Cara Menggunakan <br />
1. Upload Sketch ESP32CAM dengan mengganti SSID dan Password WiFi <br />
2. Menjalankan program gunakan: <br />
```
python kalori.py --modeldir=tflite
```
